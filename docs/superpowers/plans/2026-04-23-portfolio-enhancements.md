# Portfolio Site Enhancements Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a home section, new project stubs, portfolio discipline filters, and build-time PDF generation to the existing single-page portfolio — without changing the existing design, resume content, or Fancybox lightboxing.

**Architecture:** All changes are additive. The home section is a new `<article id="home">` tab in the existing SPA nav. Portfolio items get `data-discipline` wrapper divs and filter buttons inside the portfolio article. PDF is generated by Puppeteer after `vite build` and served at `/pdf` via a redirect page in `public/`.

**Tech Stack:** Vanilla HTML/CSS/JS, jQuery 3.3.1 (existing CDN), Fancybox 3.3.5 (existing CDN), Vite 5, Puppeteer (new devDependency), Node.js http module for PDF generation static server

## Context: How the existing site works

`index.html` is a single-page app. The `<section>` has `flex-direction: column-reverse` but that only matters for visual stacking when multiple articles are shown — since only one article is shown at a time via JS, it doesn't affect the feature work here.

`navctrl(name)` hides all `<article>` elements, removes `.selected` from nav links, then shows the named article and marks the matching nav link as selected. The `init()` function currently calls `navctrl('resume')` — we'll change this to `navctrl('home')`.

Fancybox 3 opens via URL hash: `<a href="#ucsf">` triggers Fancybox to open `[data-fancybox="ucsf"]` gallery via its built-in hash feature. Hidden `<a>` elements with `data-fancybox="GROUP"` are still registered by Fancybox even when `display:none`.

Portfolio images are at `../media/NAME.jpg` relative to `index.html` at root (e.g. `../media/ucsf-0.jpg` → `/media/ucsf-0.jpg` since going "up" from root stays at root).

```js
function filterPortfolio(discipline) {
    $('.portfolio-item').each(function() {
        var d = $(this).data('discipline') || '';
        var disciplines = d.split(' ');
        if (discipline === 'all' || disciplines.indexOf(discipline) !== -1) {
            $(this).show();
        } else {
            $(this).hide();
        }
    });
    $('.filter-btn').removeClass('selected');
    $('.filter-btn[data-filter="' + discipline + '"]').addClass('selected');
}
```

- [ ] **Step 2: Create scripts/ directory and generate-pdf.js**

Create the file `scripts/generate-pdf.js` with this complete content:

```js
import http from 'http'
import fs from 'fs'
import path from 'path'
import { fileURLToPath } from 'url'
import puppeteer from 'puppeteer'

const __dirname = path.dirname(fileURLToPath(import.meta.url))
const distDir = path.resolve(__dirname, '../dist')
const outputPath = path.join(distDir, 'resume.pdf')

const MIME = {
  '.html': 'text/html',
  '.css': 'text/css',
  '.js': 'application/javascript',
  '.jpg': 'image/jpeg',
  '.jpeg': 'image/jpeg',
  '.png': 'image/png',
  '.gif': 'image/gif',
  '.ico': 'image/x-icon',
  '.svg': 'image/svg+xml',
  '.woff': 'font/woff',
  '.woff2': 'font/woff2',
}

function serveDir(dir) {
  return http.createServer((req, res) => {
    let urlPath = req.url.split('?')[0]
    if (urlPath === '/') urlPath = '/index.html'
    const filePath = path.join(dir, urlPath)
    const ext = path.extname(filePath)
    try {
      const data = fs.readFileSync(filePath)
      res.writeHead(200, { 'Content-Type': MIME[ext] || 'application/octet-stream' })
      res.end(data)
    } catch {
      res.writeHead(404)
      res.end()
    }
  })
}

async function main() {
  const server = serveDir(distDir)
  await new Promise(resolve => server.listen(0, '127.0.0.1', resolve))
  const port = server.address().port

  const browser = await puppeteer.launch({
    args: ['--no-sandbox', '--disable-setuid-sandbox'],
  })
  const page = await browser.newPage()

  await page.goto(`http://127.0.0.1:${port}/`, { waitUntil: 'domcontentloaded', timeout: 30000 })
  await page.waitForFunction(() => typeof navctrl !== 'undefined')
  await page.evaluate(() => navctrl('resume'))
  await page.waitForFunction(() => {
    const el = document.getElementById('resume')
    return el && el.style.display !== 'none'
  })

  await page.pdf({
    path: outputPath,
    format: 'Letter',
    printBackground: false,
    margin: { top: '0.75in', bottom: '0.75in', left: '0.75in', right: '0.75in' },
  })

  await browser.close()
  server.close()
  console.log(`PDF written to ${outputPath}`)
}

main().catch(err => { console.error(err); process.exit(1) })
```
